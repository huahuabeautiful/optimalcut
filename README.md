# Optimal Segment v4.1 - 遥感影像数据集裁剪与处理算法

本项目是一个用于高分辨率遥感影像（如 GF1、GF6 卫星数据）深度学习数据集构建的自动化预处理与裁剪工具。特别适用于需要从大尺度的遥感影像及 Shapefile 矢量文件中，自动提取、裁剪并生成高质量语义分割数据集的场景（例如侵蚀沟等地理特征的自动化提取）。

## ✨ 核心功能 (Features)

*   **规则化网格裁剪**：支持基于规则网格对大幅遥感影像进行切片计算与生成，避免内存溢出。
*   **高精度矢量解析**：支持直接读取并处理 Shapefile (.shp) 文件，并将其精准映射到裁剪后的影像块中作为深度学习的标签 (Mask)。
*   **折线骨架提取**：内置骨架提取算法，能够将面状或带状矢量目标转化为折线（Polyline）中心线，满足特定网络对线状特征的训练需求。
*   **无效数据过滤优化**：在切片生成过程中，自动过滤纯黑背景或无有效目标区域的废片，确保输出数据集的高质量，提升模型（如 DeepLabV3+、U-Net 等）的训练效率。

## 📂 项目结构 (Project Structure)

*   `main.py`：主程序入口，负责协调各个模块的运行。
*   `config.py`：全局配置文件，包含输入输出路径、裁剪尺寸、无效图像过滤阈值等参数。
*   `shp_processor.py`：矢量文件处理器，负责 Shapefile 的读取、坐标系对齐与解析。
*   `mask_generator.py`：标签生成器，根据矢量信息在对应的影像切片上绘制二值化或多类别的 Mask 图像。
*   `slice_calculator.py`：切片计算器，负责根据原始影像尺寸和目标切片大小（及重叠率）计算网格坐标。
*   `slice_generator.py`：切片生成器，执行实际的影像裁剪操作。
*   `slice_optimizer.py`：切片优化器，负责对生成的切片和标签进行质量检验与清洗（如剔除全黑底图）。
*   `skeleton_extractor.py`：骨架提取器，用于从生成的标签或矢量中提取目标的折线特征。

## 🚀 快速开始 (Quick Start)

1.  **环境配置**：
    确保你的 Python 环境中已安装必要的遥感与图像处理库（推荐使用 `geopandas`, `rasterio`, `opencv-python`, `shapely` 等）。

2.  **修改配置**：
    打开 `config.py`，根据你的本地环境设置：
    *   原始高分影像文件路径。
    *   对应的目标区域 Shapefile 文件路径。
    *   期望的切片尺寸（例如 512x512）及步长。
    *   输出数据集的保存目录。

3.  **运行程序**：
    配置完成后，直接运行主脚本即可开始批量生成数据集：
    ```bash
    python main.py
    ```

## 📝 适用场景
本算法专为复杂的地理空间数据预处理设计，生成的标准格式数据集可直接对接 PyTorch 等深度学习框架的 DataLoader，用于后续的语义分割模型训练与预测。
